Colorette

A simple color palette generator that lets you save and share your color palettes.

  • ✅ Try out the colors directly on the site
  • ✅ Save any created palettes in local storage
  • ✅ Save palettes in URL so they can be easily shared
  • ❌ Show saved palettes in a popup (button in place)
    • ❌ Show the palette with all colors as swatches
    • ❌ Hover over to change style on page, but not update the input values
  • ❌ Redo background and add more backgrounds
  • ❌ Add a tab/second view with all colors clearly visible by hex value etc
  • ❌ Use web components and events
  • ❌ Atproto support? https://atproto.com/
